flame_utils.misc.listset.get_intersection(**kws)[source]¶

Get the intersection of all input keyword parameters, ignore empty list or tuple.

Returns
reslist

Examples

>>> a, b, c = [], [], []
>>> print(get_intersection(a=a,b=b,c=c))
[]
>>> a, b, c = [1], [2], []
>>> print(get_intersection(a=a,b=b,c=c))
[]
>>> a, b, c = [1,2], [2], []
>>> print(get_intersection(a=a,b=b,c=c))
[2]
>>> a, b, c = [1,2], [2], [2,3]
>>> print(get_intersection(a=a,b=b,c=c))
[2]
>>> a, b, c = [1,2], [3,4], [2,3]
>>> print(get_intersection(a=a,b=b,c=c))
[]
